# Portfolio Web - Rafael Navarro

Portfolio personal en HTML y CSS con proyectos de clase orientados a maquetación, estructura semántica y diseño responsive.

## Estructura

- `Portfolio.html`: página principal del portfolio
- `stylos.css`: estilos globales
- `Carta-bar/`: proyecto de carta de bar y formulario de pedido
- `TechSolutions/`: mini portal corporativo

## Proyectos incluidos

### Carta de Bar
- Inicio: `Carta-bar/index.html`
- Pedidos: `Carta-bar/pedido.html`

### TechSolutions
- Inicio: `TechSolutions/index.html`
- Recursos: `TechSolutions/videos.html`

## Tecnologías

- HTML5
- CSS3
