Portafolio Personal - Rafael Navarro Rosa

Este proyecto es un portafolio web personal diseñado para mostrar la identidad profesional, proyectos y habilidades de Rafael Navarro Rosa, estudiante de Desarrollo de Aplicaciones Multiplataforma (DAM).

La web presenta un diseño moderno, colorido y completamente responsivo, facilitando la navegación tanto en dispositivos móviles como en escritorio.

🚀 Características Principales

Sección "Sobre Mí": Breve introducción profesional con enlaces directos a redes sociales.

Galería de Proyectos: Espacio dedicado a mostrar trabajos previos, incluyendo descripciones detalladas y tecnologías utilizadas.

Formulario de Contacto: Interfaz funcional para que posibles reclutadores o colaboradores envíen mensajes.

Diseño Responsivo: Adaptación fluida mediante Media Queries para pantallas de diferentes tamaños (móviles, tablets y PC).

Navegación Suave: Implementación de scroll-behavior: smooth para una mejor experiencia de usuario al desplazarse por las secciones.

🛠️ Tecnologías Utilizadas

HTML5: Estructura semántica del contenido.

CSS3: Estilos personalizados, incluyendo:

Gradientes lineales dinámicos.

Flexbox para la alineación de elementos.

Animaciones interactivas en botones (keyframes).

JavaScript: (Mencionado como tecnología base en el proyecto de "Gestor de Tareas").

📂 Estructura del Proyecto

Plaintext

├── Portafolio.html  # Estructura principal de la web

└── stylos.css       # Hoja de estilos con diseños y animaciones

💻 Proyecto Destacado

Gestor de Tareas Web

Una aplicación interactiva para la organización diaria que permite:

Crear, editar y eliminar tareas.

Marcar tareas como completadas.

Almacenamiento local de datos para persistencia.

Tecnologías: HTML5, CSS3, JavaScript.
