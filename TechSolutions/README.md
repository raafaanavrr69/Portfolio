# TechSolutions

Sitio web corporativo estático orientado a presentar servicios tecnológicos, horario de atención y recursos de formación.

## Contenido

- `index.html`: página principal con presentación, horario y sección de noticias
- `videos.html`: recursos de formación con vídeos embebidos
- `styles.css`: estilos de la web

## Tecnologías

- HTML5
- CSS3

## Objetivo del proyecto

Practicar maquetación semántica, navegación entre páginas y diseño visual para un portal corporativo.
